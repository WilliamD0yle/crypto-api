A simple react app using the crypto compare api to display crypto coins price data.
[https://github.com/WilliamD0yle/crypto-api](https://williamd0yle.github.io/crypto-api/#/)
### Installation
`yarn install`

or 

`npm install`

### Run webapp
`yarn start`

or

`npm start`

### Testing
To run test: `yarn test` or `npm run test`
